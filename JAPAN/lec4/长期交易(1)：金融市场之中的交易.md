
# 导论1 金融系统的功能与经济发展
3.25课程 开头介绍：多个主体之间的关系

**单纯循环经济+政府+金融+创新**
创新离不开金融。创新有两种：产品创新与工艺创新；金融有两种：直接金融与间接金融。
直接金融是资本市场依存性（股票债券），间接金融是贷款市场依存性（银行贷款）

金融系统的职能：
- 经济资源时空转移（借贷、期货）
- 控制经营风险（保险、对冲：与其它国家贸易的时候，处理汇率变化问题）
- 方便商业结算（信用证）
- 为经济决策提供金融价格信息
- 为经纪人或者代理人提供激励（作为薪酬的期权，让你在干活多少年之后可以获得公司对应的股权）

# 导论2 信息不对称与金融机构的分布
>日本报纸订阅量世界第一。
	关注经济类的报纸。
	英美经济类报纸更强调**金融**，而德日则没有这么明显。


为什么不同国家中金融机构比例不同？

有哪些金融机构？
- 银行
- 投行：大多都是美资
- 四大会计师事务所：基本都是美资
- 评级公司
- 咨询公司
除了银行之外，剩下基本上都是被美资控制。因此金融机构可以被划分为银行和非银行。二者区别在于能否创造货币，银行可以，非银行不可以。

*如果信息对称了，那么利润就消失了*

*信息不对称是市场失败的表现；引发了逆选择、道德风险问题*

- 逆选择：劣币驱逐良币，合约没签约呢，最后只能签约不好的客户
- 道德风险：不好的客户知道自己可以获利，因此开始放纵自己；低风险承诺，高风险实操

>三个账本：税务局看的、贷款时候给银行看的、自己看的

>如何解决信息不对称问题？

- 美国：本质上非银行金融机构卖的是金融信息；每个类型信息都有不同的银行相互竞争、相互售卖信息。让不对称的金融信息变成商品，在复数的机构竞争中逐渐提升，最终逐步实现信息的对称。
- 德日：银行锁定个人，长期反复交易。在反复交易过程中，银行逐渐对客户了解更深入，实现信息对称。

# 导论3 法人企业两重性问题
法人：把没有人格的东西拟人化

股东拥有企业：作为物的企业
企业拥有资产：作为人的企业

企业本质不同观点：
1. 一组契约的集合（可以通过市场交易来重组）（企业本质上是虚体）
2. 一组资产的集合 （本质是实体）

这就决定了不同的企业治理机制
1. 企业进行战略决策的机制
2. 调节利益相关者利益的机制 - 核心利益相关者和周边利益相关者（最大问题：核心利益相关者之间的利益并不总是一致的）
3. 对管理层进行监管的机制 （实际所有者如何控制管理层进行战略决策）
>所有者监督管理者进行战略决策、调节相关者利益的机制

企业治理机制可以分为两种类型：股东主权、从业员主权。（回溯到企业归谁所有这个问题）（从业员也可能隐含某个企业的主题内容）

# 高度成长期日本金融市场的特征
- 金融监管当局
	- 维持信用秩序
	- 促进经济发展，低利率政策，降低企业融资成本；但对人民不是好事
	- 抑制债券市场的发展，把资源引到银行去
- 企业金融
- 企业治理结构：从股票价格和并购的压力下解放出来，受限于主力银行

通过降低利率，把金融风险社会化，缓解不同金融机构的对立，形成了以主力银行为主体的企业治理机构。

# 交叉持股与主力银行
日本经济中的人质效果。各个主体之间都有人质效果。

交叉持股表面上收益大于成本（交易成本降低，风险共担，信息交换）
但是在深层次上，账面价值高但是资本效率不高；沉默的股东和偶尔混乱的股东大会。

主力银行是企业的最主力融资来源。进行信息创造和传递功能；救济功能。

>对与家庭：金融资产不会给他带来很大的收益
