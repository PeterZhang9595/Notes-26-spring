# testing
很多时候我们使用简单的测试方式无法覆盖所有可能出现的情况，因此使用形式化验证方式是很重要的，即考虑所有可能的输入方式。
当然我们在工业生产上很难完全实现成本和难度极高的形式化验证，但是我们仍然可以选择全面且系统化的测试样例。
