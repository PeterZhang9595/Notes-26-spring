# Bits
信息是以比特为单位进行测量的。
比特的总数是所有可能出现的结果的对数量级。

在日常生活中，我们对某件事情的不确定性、信息的一种缺失，就可以被以bit为单位进行表示。

## 量子比特
根据量子物理理论，对于任何一个小物体，它是有可能存在两个可以被测量的状态的。被称为**qubit**,符号表示为$|0\rangle,|1\rangle$，而一个qubit可以表示为$\alpha |0\rangle + \beta |1\rangle$。

### 可逆性
如果一个状态可以通过某种转换的方式成为另外一个状态，那么相反的转换方式也是有可能的。

然而在量子系统里面，也存在一些显然的不可逆性质：
- 量子系统必定会与未知状态的周围环境产生交互，导致系统的信息发生丢失。
- 对系统的测量行为是不可逆的。

### 叠加态
测量量子的状态只可能出现两种结果0或者1。每一次测量的结果都无法被预测，然后每一次出现不同的结果它们分别的概率是确定的。

量子的偏振反向可以用于存储一个bit的信息。

>为什么一个量子只能用于表示一个bit，难道偏振角难道不可以表示更多bit信息吗？
因为在量子信息里面，我们无法像在现实中那样拿过来用量角器测量。我们只能预先选定一个一个测量角度，然后量子过来我们问一下：你是我要测的角度吗？然后量子会回答0或1，仅此而已。
如果是1，那么就是我们预先定好的角度，如果是0，那么就是垂直于预先定好的角度。

>为什么不能重复测量？
测量后量子坍塌。怎么测都是一个结果了。

### 量子纠缠
两个看似相互独立的量子也存在某种纠缠的状态。

### qubit的好处
情景：A和B进行量子通信。A准备了垂直和水平的量子，并且告诉B对应的测量标准。

然而在传输的过程中，有一个SB在搞破坏。由于测量会导致量子坍塌，所以SB随便选了一个测量角度，破坏了原有的角度。然后B接到信息开始解码，但是解码过程中发现不对劲，0和1的概率几乎是1：1。那么他就可以重新和A交流。

## 经典比特
在传统的比特实现中，使用的是连续电压，它具有以下的好处：
- 可以对电压值设定阈值，抗噪声能力强
- 测量不会影响它的状态

但是如果在更小的器件上，它能否仍然发挥效果有待观察。
