# 美国卫生与公共服务部的历史
- 联邦政府成立之初，高度依赖关税与海洋运输，但是水手大量被感染传染病。为了给水手治病，设立了海事医院服务局，成为美国最早的联邦公共卫生服务机构。
- 1889年成立公共卫生服务军官团，以军队建制建立全国传染病监测与防控体系，成为美国八大军种之一。
- 1930年代，联邦政府重组卫生、教育、福利三大体系，纳入联邦保障署统一行政架构，确立联邦政府在全国公共卫生体系中的核心地位
- 二战结束后，把”卫生、教育与福利部“正式升级为内阁正部级。
- 1960年代，为了与苏联进行竞争，加大社会保障程度，提高公共医疗体系与社会福利程度
- 1980年，被拆分为教育部和美国卫生与公共服务部

>公共卫生服务军官团

- 解决了联邦制分权管理与公卫需要集中应对的制度缺陷。
- 美国医生执照由州卫生厅颁发的。因此无法把全国的医生进行统一安排。
- 因此要成立可以集中安排的部队来应对威胁国家安全的公共卫生危机，军团便快速响应。

>美国国立卫生院

全球规模最大的生物医学研究机构。标志从纯粹的传染病管控转向科学研究。

- 美国国立卫生院是美国高校医学院的核心经费来源。
- 美国顶级名校医学院是自负盈亏的科研运营实体，绝大多数教研人员实行合同制聘用，受聘首要条件还需要研究团队具备充足的外部科研经费，“带薪进校，付费上班”
- 哈佛大学医学院与研究团队之间是共赢共生的生态结构。内卷式学术资本主义很严重。

**NIH是中美科技脱钩的急先锋。针对中国进行了很多科研上的数据限制等等**

>联邦食品药品管理局FDA

早期食品药品乱象很多，香肠哈哈哈。
- 处方药的起源：FDA明确要求部分药物应该在专业医师的监督下使用。prescription drug
- 非处方药：OTC

- FDA高光时刻
莎莉度胺事件。缓解孕吐的药，制药厂想要在美国上市，FDA科学家认为药厂提供的数据只是提供了治疗的有效性但是没有证明安全性。由于FDA最终没有批注，保证了美国孕妇和婴儿的安全，因此FDA获得了高光时刻，为女科学家颁奖。

- 1991年，FDA由于官员被贿赂的丑闻，宣布成立刑事调查办公室。

>美国疾控中心 CDC

不仅是疾病控制与预防中心，更是一个准军事化体系。

一开始就是给军队作服务的，完善后期保障。
- 喝热水：建国之后的事情。朝鲜战争期间，美军在东北投放大量细菌弹。因此为了防治疾病流行，开始流行喝热水。

流行病情报局检测可能的人为恐怖袭击。由于精英化选拔、军事化部署，保留准军事化的组织特征。

CDC在全球许多国家开设了办事处，2003年在北京设立了办公室。
爆发非典之后，中国防疫站改革，和美国疾病预防控制中心大概一样，转向信息收集。

但是这个傻鸟CDC在美国的风评很不好。
**塔斯基地梅毒实验，测试各种药品实验；甚至在危地马拉，让健康的囚犯感染性病病毒，然后做不同药物效果对比试验**
**猪流感疫苗计划**
这也就导致了美国人对于疫苗的极度不信任。
