# 基础发展历史
1. 殖民地时期：宗教主导与教育萌芽。
清教徒在马萨诸塞州建立殖民地后，实行政教合一。强制宗教顺从。
早年的各种grammar school的作用就是给各种教堂培训神职人员。

南方的社会结构分化明显，社会教育发展比较落后。

2. 公立学校教育的扩展：从宗教到国民身份构建
1830年后，美国开始快速工业化，对于学校的需求产生重大变化，需要学生具有初步识字能力、懂得合作、懂得遵守法律遵守规则。
此时的劳动力需求量变大了很多，因此推动了"common school movement"。
- 所有学童都应该接受机会均等的免费教育
- 教育的目标是摆脱宗教控制
- 根据学童年龄而非天赋，实行年级进阶制，职业化教师按照统一教材进行标准化教学

1852年，马萨诸塞州颁布《义务教育法》，规定8-14岁儿童必须入学接受教育。
1892年，成立美国教育联合会，制定了K12教育制度，标志着美国标准化基础教育体系的正式确立。

**在美国这个移民国家，教育，就是建立移民之间相互认同的唯一方式。这也成为了教育最重要的政治功能**

*美国梦？就是通过教育灌输到美国公民的脑袋中的。每天都要升旗，唱国歌，念誓词*

一定要关注民族国家认同的构建方式。这对于中国来说，由于有较多的民族共同记忆，因此还是相对容易的。

3. 南方废奴运动后的教育体系
内战后，南方出现了白人与黑人以种族为界限的“分离式”教育现象。
这是源于早年白人对黑人奴役，而黑人对白人形成的不信任的心理。

19世纪末，南方各州建立了两套平行的公立教育体系。
当然，当时白人和黑人在各个方面都受到种族隔离法的影响，很多事情都是被分割开的。

1951布朗案，来时取消种族隔离。
1957年“小石城事件”联邦政府派遣101空降师（美国二战期间最精锐的部队），保护黑人入学。后面1964美国颁布《民权法典》，禁止在教育领域实施种族歧视，从此美国学校的种族融合才取得实质性进展。

4. 改革与多元化发展
- 20世纪初，受约翰杜威影响，主张教育应该服务于学生的生活与社会需求，注重培养实践能力与创新精神
- 1957年苏联发射第一颗人造卫星，导致美国进步教育运动模式的衰落。联邦政府首次大规模干预教育，强化理工科教育，培养掌握先进科技的高精尖人才。
- 高强度选拔导致大量学生出现心理问题，为家庭带来焦虑和医疗负担，加上国家安全威胁减弱，“后工业化”思潮兴起，美国转向“快乐教育”，即“情感学习”或者“人本主义教育”重新兴起。
